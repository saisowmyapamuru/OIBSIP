# Voice Assistant

## Project Overview

Voice Assistant is a Python-based application that responds to voice commands. It can greet the user, tell the current time, provide the current date, and respond to basic commands using speech recognition and text-to-speech technologies.

## Features

- Voice command recognition
- Text-to-speech response
- Current time announcement
- Current date announcement
- Greeting commands
- Exit command

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Datetime
